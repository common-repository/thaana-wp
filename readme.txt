Thaana keyboard support for Posts/Pages. Created using JTK by Jawish Hameed (jawish.org).

== Description ==

Enables support for Thaana keyboard on Post/Page editors. Change keyboard types at Thaana Wp Settings page under Plugins.
Created using Jaa's <a target="_blank" href="http://www.jawish.org/blog/archives/436-Integrating-JTK-with-tinyMCE-4.x.html">Integrating JTK with tinyMCE 4.x</a> AND <a target="_blank" href="https://github.com/jawish/jtk">JTK</a>

== Installation ==

Upload the Thaana WP plugin to your Wordpress plugins directory, Activate it. Change the settings for the keyboard or to disable, go to Plugins >  Thaana Wp.
